# OpenMeow
OpenMEOW is a modular desktop toolset inspired by PowerToys, infused with cat energy and modern design. It's a free, open-source utility app built for developers, designers, and productivity nerds who want a bit more comfort on their desktop.

## Features
- Color Picker — instantly grabs the color under your cursor.
- JSON ↔ YAML Converter — fast, clean, and reliable conversion.
- Ruler — a screen ruler for UI/UX perfectionists.
- Modular system — easily extendable with custom utilities.
- Cute cat-themed interface. Because why not?

## Installation
1. Install Node.js and npm.
2. Clone the repository: git clone https://github.com/FedotovDev/openmeow
3. Navigate to the folder: cd openmeow
4. Install dependencies: npm install
5. Launch the app: npm start
